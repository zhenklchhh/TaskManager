### Доступные метрики

#### HTTP метрики
- `http_requests_total` - общее количество HTTP запросов
- `http_request_duration_seconds` - длительность HTTP запросов

#### Метрики задач
- `tasks_total` - общее количество задач
- `task_processing_duration_seconds` - длительность обработки задач

#### Метрики базы данных
- `db_connections_active` - активные соединения с БД
- `db_query_duration_seconds` - длительность запросов к БД

#### Метрики Redis
- `redis_connections_active` - активные соединения с Redis

#### Метрики планировщика
- `scheduler_runs_total` - количество запусков планировщика
- `scheduler_errors_total` - количество ошибок планировщика
